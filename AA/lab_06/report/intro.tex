\ssr{ВВЕДЕНИЕ}
 
Целью данной лабораторной работы было исследование алгоритмов решения задачи коммивояжёра: алгоритма полного перебора,
муравьиного алгоритма.
 
 Для достижения поставленной цели необходимо было выполнить следующие задачи:
 \begin{enumerate}
 	\item описать метод полного перебора для решения задачи коммивояжёра; 
 	\item описать метод решения задачи коммивояжёра на основе муравьиного алгоритма; 
 	\item реализовать оба алгоритма;
 	\item выполнить оценку трудоёмкости составленных алгоритмов по разработанным схемам алгоритмов;
 	\item выполнить параметризацию последнего метода;
 	\item дать рекомендации о значениях либо диапазонах значений параметров, настройках и/или режимах работы метода, основанного на муравьином алгоритме.
\end{enumerate}
