\ssr{ВВЕДЕНИЕ}
 
Цель: разработка и реализация конвейерной обработки набора заявок на кластеризацию вершин графа по алгоритму DBSCAN.

 Для достижения поставленной цели необходимо было выполнить следующие задачи:
 \begin{enumerate}
 	\item описать последовательный алгоритм DBSCAN; 
 	\item разработать параллельную версию алгоритма; 
 	\item реализовать обе версии алгоритма;
 	\item реализовать последовательную обработку $N$ заявок;
 	\item разработать параллельный конвейер обработки заявок, включающий три обслуживающих устройства, взаимодействующих через очереди;
 	\item реализовать конвейер;
 	\item провести замеры времени обработки $N \in \{25, 50, 75, 100, 125\}$ заявок для последовательной и конвейерной реализаций.
\end{enumerate}
