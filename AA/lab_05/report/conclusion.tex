\ssr{ЗАКЛЮЧЕНИЕ}

В результате лабораторной работы была разработана и реализована конвейерная обработка набора заявок на кластеризацию вершин графа по алгоритму DBSCAN.

Выполнены следующие задачи:
\begin{enumerate}
	\item описан последовательный алгоритм DBSCAN; 
	\item разработана параллельная версия алгоритма; 
	\item реализованы обе версии алгоритма;
	\item реализована последовательная обработка $N$ заявок;
	\item разработан параллельный конвейер обработки заявок, включающий три обслуживающих устройства, взаимодействующих через очереди;
	\item реализован конвейер;
	\item проведены замеры времени обработки $N \in \{25, 50, 75, 100, 125\}$ заявок для последовательной и конвейерной реализаций.
\end{enumerate}
